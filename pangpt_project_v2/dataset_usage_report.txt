4. HOW WINDOWING WORKS
-------------------------------------------
Example: If a genome has 1000 genes and window size = 256:

  Window 1:  genes[0:256]     → positions 1-256
  Window 2:  genes[256:512]   → positions 257-512
  Window 3:  genes[512:768]   → positions 513-768
  Window 4:  genes[768:1000]  → positions 769-1000 (remainder, padded)

  Result: 4 windows from 1 genome

Key points:
  ✓ Each genome is split into multiple windows
  ✓ Windows don't overlap (shift = window_size)
  ✓ Last window may be shorter (gets padded)
  ✓ Gene order is preserved within each window
